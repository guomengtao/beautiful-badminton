# Beautiful Badminton 🏸

一个使用 React、TypeScript 和 Canvas 构建的现代交互式羽毛球游戏。与 AI 对手对战或在本地多人模式中挑战朋友！

## 功能特点

- 🎮 两种游戏模式：玩家对战机器人和玩家对战玩家
- 🎨 精美的响应式设计和流畅的动画
- 🌟 真实的物理模拟
- 🏆 比分追踪
- 🎵 直观的控制和游戏玩法

## 开始使用

1. 克隆仓库：
   ```bash
   git clone git@github.com:guomengtao/beautiful-badminton.git
   ```

2. 安装依赖：
   ```bash
   npm install
   ```

3. 启动开发服务器：
   ```bash
   npm run dev
   ```

4. 打开浏览器访问 `http://localhost:5173`

## 游戏操作

### 控制方式
- **玩家1:** 使用左右方向键
- **玩家2 (多人模式):** 使用 A 和 D 键

### 游戏模式
- **对战机器人:** 挑战AI对手
- **对战玩家:** 双人本地对战模式
